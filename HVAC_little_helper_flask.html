HVAC Little helper Flask

The first iteration of the HVAC applications was created to make all the offer requests faster and safer. I was doing that at my then-current job, and it was a manual task that was slow and prone to error. THe app was created using Flask and Jinja, storing everything in a PostgreSQL database. Main functionality was automatic offer creation based on the existing catalog. There were frequent items requested and I made a way to calculate the price of each item based on their dimensions.

Features:
  1. Automatic cost calculation
  2. Offers created instantly
  3. Creation of personalized labels for each product
  4. Tracking of consumed raw materials
  5. More than 50 endpoints
  6. Persistent data using a PostgreSQL database

Technologies

  1. Python
  2. PostgreSQL

Python libraries used

  1. Flask
  2. Raw SQL
  3. PyPDF
  4. Pandas
