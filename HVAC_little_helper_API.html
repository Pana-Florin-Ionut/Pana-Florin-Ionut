HVAC Little helper API

HVAC little helper API represents the second iteration of an app that was designed to help the HVAC industry (of, if I am honest, all the industries that are working with offer requests) to make all the offer requests faster and safer. I was doing that at my previous jobs, and it was a manual task that was slow and prone to error.
Features:
  1. Token-based user authentication
  2. Separate profiles for Companies and users
  3. Users can be independent or belong to a company
  4. More than 50 endpoints
  5. Separate projects and offers
  6. Thought from the ground up for the interaction between clients and sellers
  7. Persistent data using a PostgreSQL database

Technologies

  1. Python
  2. PostgreSQL

Python libraries used

  1. FastAPI
  2. SQL Alchemy
  3. OAuth2
  4. Pydantic
